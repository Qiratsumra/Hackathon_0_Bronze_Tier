# Brain Log

This directory contains the agent's reasoning logs and decision records.

## Structure

Each decision is logged as a markdown file with:
- **Timestamp**: When the decision was made
- **Decision**: The action or choice made
- **Reasoning**: The agent's reasoning process

## Example Log Entry

```
# Agent Decision

**Time**: 2026-01-17T10:30:00

**Decision**: Send email to team about project status

**Reasoning**: 
The agent detected an incoming email requesting project updates. 
Based on the task context and available information, sending a 
comprehensive status email is the most appropriate response.
```

## Analysis

Use this log to:
- Understand agent behavior
- Debug decision-making
- Audit automated actions
- Identify patterns

See recent decisions in the parent vault dashboard.
